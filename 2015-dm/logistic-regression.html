공지사항

  1. 이 수업은 프로그래밍 수업이 아니라 데이터마이닝 수업입니다
    • 프로그래밍 팁을 공유하고 파이썬 튜토리얼을 병행하는 것은 어디까지나 여러분을 위해 하는 것
      • 프로젝트 할 때 막막하지 말라고
      • 이론에 그치는 데이터마이닝이 아니라 실질적인 마이닝 능력을 키우라고
      • 프로그래밍 언어 하나라도 자유자재로 쓸 수 있게 돼서 어디 가서 써먹으라고
    • 프로그래밍? 시험에도 안 나옴 (혹~시라도 나오게 된다면 사전 공지)
    • 하지만 프로그래밍은 여러분에게 큰 자산이 될 것이라고 확신
    • 개인 경험: 프로그래밍 입문 (C 언어), C $to$ Matlab $to$ Java $to$ Python
    • 팁: 하나의 언어를 제대로 익힌 후 다른 언어를 배우면 훨씬 쉽다
