Multiple linear regression (revisited)

  • 지난 시간에 우리는 MLR(multiple linear regression)에 대해 배우고 파이썬으로 실제 데이터에 대해 모델을 구축해보았다
  • 그런데 우리 컴퓨터는 MLR을 어떻게 학습했을까? (i.e., 파라미터를 어떻게 추정했을까?)
  • 이번 시간에는 MLR의 블랙박스(black box) 안에 무엇이 있는지를 보자.

  • MLR: Fit a linear relationship between a quantitative dependent variable $y$ and a set of independent variables $X$ (=$[x_1, x_2, ..., x_m]$), such that

    $$y = b_0 + b_1x_1 + b_2x_2 + ... + b_mx_m$$

    • $y$: output variable / dependent variable
    • $x_j$: $j$th input variable / independent variable
    • $b_j$: parameters / coefficients
    • $m$: Number of input variables/features

OLS(ordinary least squares)

  • MLR의 파라미터 $b_j$들을 추정하기 위해서는 OLS를 사용한다
  • 여러 개의 변수가 들어가면 복잡해지니까 단변수의 simple linear regression을 가정해보자

$$y = b_0 + b_1x_1$$

  • Formulation
    • Hypothesis: $f(x) = b_0 + b_1x_1$
    • Parameters: $b_0, b_1$
    • Cost function: $J(b_0, b_1) = \frac{1}{2n}\sum_{i=1}^{n}(f(x_1^{(i)}) - y^{(i)})^2$
      • The MSE(mean squared error, 편차의 제곱의 평균)
    • Goal: $arg \min J(b_0, b_1)$
      • 비용 $J$를 최소화하는 것이 목표
  • Solution
    • Numerical solution: Gradient descent
    • Analytical solution: Normal equation
      • Let $y = b_0 + b_1x_1 = X\beta$, then $$\beta = (X^TX)^{-1}X^Ty$$
